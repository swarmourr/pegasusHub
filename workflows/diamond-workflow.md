# Title : diamond-workflow

## Description
The Diamond Workflow is the fundamental Pegasus example, representing a four-node graph that demonstrates data dependency and parallel execution.

## File Description
* **plan.sh:** Consists of all commands to be executed to run the workflow.
* **workflow_generator.py:** Creates the abstract workflow.
* **bin Folder:** Contains executables: `findrange`, `preprocess`, `analyze`.

## How to run?
```bash
./workflow_generator.py
./plan.sh workflow.yaml
```
